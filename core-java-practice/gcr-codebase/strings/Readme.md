# Strings Exercises

Collection of Java exercises focused on String manipulation and related demos.

## Contents

- `Anagrams.java` — Check if two strings are anagrams
- `BMICalculator.java` — BMI example using string input
- `Array_Index_Out_Of_BoundsDemo.java` — Demo of index errors
- `WordLength2D.java` — Word length operations with 2D arrays
- `VotingEligibility.java` — Voting logic from string input
- `Upper_case_Compare.java`, `Lower_case_Compare.java` — Case comparisons
- `UniqueCharacters.java` — Find unique characters
- `Trim_the_Spaces.java` — Trim and normalize whitespace
- `SubStringCompare.java`, `StringCompare.java` — Substring and string comparisons
- `Split.java`, `SplitCompare.java` — Splitting strings
- `Shortest_Longest_Word.java` — Find shortest/longest words
- `Palindrome_number.java` — Palindrome checks
- `FirstNonRepeating.java` — First non-repeating character
- `CharacterFrequency.java`, `Frequency_Count_using_rray.java` — Frequency counts
- `NullPointerDemo.java`, `Illegal_Argument_Demo.java` — Exception demos
- And more utility and demo classes

## How to run

1. Compile: `javac FileName.java`
2. Run: `java FileName`

> Tip: Look at the comments at the top of files for expected input and behavior.
