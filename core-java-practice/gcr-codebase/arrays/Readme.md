# Arrays Exercises

Collection of Java exercises demonstrating array usage (1D, 2D, and related problems).

## Contents

- `Array_Sum.java` — Sum elements of an array
- `Array2DTo1D.java` — Flatten 2D array to 1D
- `BMICalculation.java` — BMI calculation using arrays
- `BMIMultiArray.java` — BMI calculations for multiple users (2D arrays)
- `Can_Vote.java` — Voting eligibility checks
- `Check_numbers.java` — Array number checks
- `DigitFrequency.java` — Digit frequency count
- `Factors.java` — Factor-related exercises
- `Fizz_Buzz.java` — FizzBuzz implementation
- `FriendsInfo.java` — Store and display friend data in arrays
- `LargestDigits.java`, `LargestDigits2.java` — Largest digit problems
- `Mean_Heights.java` — Average/mean height calculation
- `Multiplication_Table.java` — Print multiplication table
- `NumberCheck.java` — Number validations using arrays
- `ReverseNumber.java` — Reverse numbers using arrays
- `Separate_numbers.java` — Separate digits from number
- `StudentGrades2D.java`, `StudentsGrades.java` — Student grades in arrays
- `VotingCheck.java` — Voting related checks
- `ZaraBonus.java` — Bonus calculation example

## How to run

1. Compile: `javac FileName.java`
2. Run: `java FileName`

> Tip: Use Java 8 or later. If a file expects input, run and follow console prompts.
